TwitterKit

A lightweight library for talking to the Twitter REST API from iOS and Mac apps.

What Is TwitterKit?

TwitterKit is a lightweight Cocoa library for communicating with the Twitter REST API. It also supports authentication with Twitter's various OAuth workflows.

Requirements

TwitterKit requires iOS 4.0 or higher or Mac OS X 10.6 or higher.

Features

  • Authentication with the Twitter OAuth web flow.
  • Authentication with the Twitter xAuth flow.
  • Communication with the Twitter REST API using either signed or unsigned requests.
  • Simple, small API.
  • No dependencies.

Including TwitterKit in Your Project

TwitterKit is small and self-contained. It depends on no other third-party libraries or frameworks. All classes and methods are prefixed so they should not conflict with anything in your project, even if you're using libraries with similar functionality (like one of the popular Objective-C OAuth libraries, for example).

For instructions on adding TwitterKit to your project, head on over to the project page.
